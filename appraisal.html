Yes, employees can initiate a self-appraisal process within the DigiSME platform, which can then be reviewed by their managers.

Absolutely. The system allows you to set up automated appraisal cycles, send reminders, and streamline the entire process.

Yes, all historical appraisal records are securely stored in the cloud system, accessible at any time for reference and analysis.
